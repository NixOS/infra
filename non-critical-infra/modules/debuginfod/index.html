https://debuginfod.nixos.org/ provides access to debug symbols and derivation source code for debuggers supporting the debuginfod protocol. Debug symbols are provided for all derivations that enable the `separateDebugInfo` flag. 

export DEBUGINFOD_URLS=https://debuginfod.nixos.org
